# Harness Attribution

LLM の出力を「単発の生成物」ではなく **ハーネス条件に依存する確率分布からのサンプル** として扱い、気象学の **イベント・アトリビューション**（反事実比較 ＋ Risk Ratio / Risk Difference / Attributable Fraction）を生成 AI に移植する、研究・実装プロジェクト。

## 中心命題

ある出力イベント `E`（例: 無根拠な断定、ハルシネーション、JSON 不正、危険コード、テスト全通過）について、特定のハーネス要因 `A`（system prompt / RAG / few-shot / tool schema / decoding params …）を**反事実的に除去・置換・摂動**し、`P(E=1)` または強度スコアがどれだけ動くかを推定する。

```
p1 = P(E=1 | H with A)        # factual harness
p0 = P(E=1 | H without/altered A)   # counterfactual harness
Risk Ratio (RR) = p1 / p0
Risk Difference (RD) = p1 - p0
Attributable / Prevented Fraction = (方向で選ぶ。methodology.md §6 参照)
```

気象と違い、反事実世界を**実機で何度も再実行できる**のが強み。一方でプロンプト改変は外科的介入にならない（長さ・位置・トークン化・注意配分が同時に動く）ため、**ボトルネックは「反事実世界のシミュレーション」ではなく「妥当な反事実ハーネスの設計」に移る**。本プロジェクトの中核はこの反事実設計と、その妥当性検証（自然変動の包絡線との比較）にある。

## 気象 ↔ 生成 AI 対応

| 気象イベント・アトリビューション | 生成 AI（Harness Attribution） |
| --- | --- |
| 熱波・豪雨などのイベント | ある種の生成結果（正答/ハルシネーション/危険回答/形式違反 …） |
| 人為的温暖化「あり」の世界 | 現行ハーネス（system prompt・RAG・few-shot・tool 込み） |
| 人為的温暖化「なし」の反事実世界 | 特定要因を除去・置換・弱化したハーネス |
| 自然変動 | サンプリング乱数・入力差・**プロンプト表現差**・応答揺らぎ |
| 発生確率 `p1`, `p0` | そのイベントが出る確率 |
| Risk Ratio / FAR | 条件が当該出力を何倍/何 pt 起こりやすく（にくく）したか |
| 強度変化 | 品質・類似度・事実性・危険度・テスト通過数の差分 |

## スコープと非スコープ

- ✅ **分布の帰属**: 「ある条件が、ある種の出力を入力集合 `D` 上で N 倍 / Δpt 起こりやすく(にくく)した」
- ❌ **個別出力の因果説明**: 「この 1 文はこの指示のこの行が生んだ」── 別レイヤー（gray-box: logprob/attention, white-box: circuit）。混同しない。
- ❌ **責任論**: 「この誤答は RAG のせい」ではなく「この条件は誤情報採用率を Δpt 上げた」。
